## Access Rules
- **Main branch**: Protected. All changes require a PR.
- **Collaborators**: Contact [@TSLST](https://github.com/TSLST) for write access.
- **Issues/PRs**: Open to all (public repos).

## Workflow
1. Fork the repo (or create a branch).
2. Test changes locally.
3. Open a PR with a clear description.
